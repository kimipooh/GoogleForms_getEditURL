# GoogleForms_getEditURL
This Google Apps Script retrieves and saves a URL for re-editing Google Forms response data.

##To use this function, follow these steps:

1. Make sure you've set up Google Forms response data to display in Google Spreadsheets.
https://support.google.com/docs/answer/139706

2. Save and run the Google Apps Script in Google Spreadsheets.

Of course, if you want to re-edit Google Forms responses, you must allow re-editing in Google Forms.
